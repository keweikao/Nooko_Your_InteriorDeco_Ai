# Speckit 範例腳本

這個目錄包含展示最佳實踐的範例腳本。

## 執行範例

```bash
# Constitution 載入範例
npx tsx .specify/examples/constitution-example.ts

# 完整任務實作流程範例
npx tsx .specify/examples/implement-task-example.ts
```

## 學習重點

1. **漸進式載入**：先摘要，後細節
2. **精確查詢**：使用搜尋而非載入全部
3. **上下文控制**：只載入必要資訊
4. **Token 意識**：隨時追蹤 token 使用量

## 最佳實踐

參考這些範例來了解：

- 如何有效使用 MCP API
- 如何最小化 token 使用
- 如何構建高效的工作流程
