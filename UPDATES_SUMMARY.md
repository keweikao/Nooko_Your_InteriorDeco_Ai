# Nooko 更新總結 - 產業回饋整合

**更新日期**: 2024-11-16
**版本**: v1.1 - 產業專業優化版

---

## 📋 本次更新概要

根據產業專家回饋，我們對客戶經理 Agent 的問卷進行了全面優化，使其更貼近實務、更具同理心，並能有效識別報價單中的潛在問題。

---

## ✨ 核心改進

### 1. 同理心設計強化

**目標**: 從消費者立場出發，建立信任感

**實作方式**:
- 每個問題新增 `empathy_message` 欄位（綠色框顯示）
- 用友善、理解的語氣說明為何需要這個資訊
- 避免冰冷的專業術語，改用親切的說明

**範例**:
```
問題：關於預算，我完全理解這是很實際的考量。您心中的預算區間大約是多少？

💬 預算規劃很重要，我會根據您的預算提供最合適的方案。
   如果預算有彈性，我也會提供不同等級的選擇讓您參考。
```

---

### 2. 靈活跳過機制

**目標**: 不強迫用戶回答不確定的問題

**實作方式**:
- 關鍵但可能不確定的問題加入 `can_skip: true`
- 提供 `skip_suggestion` 說明為何可以跳過
- 前端顯示「稍後再決定」按鈕
- 點擊後自動填入「稍後由設計師現場確認」

**可跳過的問題**:
- 房屋狀況（如果不清楚漏水壁癌等）
- 垃圾清運費（報價不明確時）
- 施工保護與清潔（報價未提及時）
- 油漆計價方式（不了解行情時）
- 油漆工法細節（專業術語不懂時）
- 壁紙去除（不確定時）
- 社區規定（尚未了解時）
- 電梯狀況（待確認時）

### 3. 產業回饋整合

基於產業專家的詳細回饋，我們調整了以下問題：

#### 🗑️ 垃圾清運費（新增）

**問題背景**:
> 產業回饋：「報價單沒有看到垃圾清運的部分就需要補足，或者請業者明訂是否贈送，避免後續爭議。」

**新增問題**:
```
Q: 您的報價單有沒有明確標示「垃圾清運費」？這是很容易被遺漏的項目。

選項：
- 報價已包含
- 報價標示為「贈送」
- 報價沒提到
- 不確定

💬 垃圾清運費常常被遺漏或標示不清。如果是「贈送」，建議還是請業者
   明訂在合約中，才能避免後續糾紛。
```

---

#### 🧹 施工保護與清潔（新增）

**問題背景**:
> 產業回饋：「是否含施工後保護、清潔？這個不太像施工順序，有點像工法確認。」

**新增問題**:
```
Q: 拆除後的現場保護和清潔，報價有包含嗎？

選項：
- 有包含施工後保護與清潔
- 只包含保護，不包含清潔
- 都沒包含
- 不確定

💬 施工保護（如電梯、公共區域）和每日清潔，都是容易被忽略但很重要的項目。
```

---

#### 🎨 油漆工程（大幅優化，1題→4題）

##### **3.1 油漆品牌與價格參考**

**問題背景**:
> 產業回饋：「塗料品牌的市場行情不一樣，ICI乳膠漆一坪約1500、班傑明一坪約3200，AI可以藉由品牌判斷收費是否過高。」

**優化問題**:
```
Q: 您的報價單有標示油漆品牌嗎？不同品牌價差很大。

選項：
- 有標示品牌
- 沒有標示，只寫乳膠漆
- 不確定

ℹ️ 目的：油漆品牌會大幅影響價格，例如 ICI 約1500/坪、班傑明約3200/坪

💬 油漆品牌的價差可以很大，了解品牌才能判斷報價是否合理。
```

##### **3.2 油漆計價方式（坪數 vs 一式）**

**問題背景**:
> 產業回饋：「油漆列為『式』，缺少面積基準，不利跨廠商比價。用『一式』可能是數量太少、工法複雜、或包含多項施工。」

**新增問題**:
```
Q: 油漆報價是用「坪數」還是「一式」計價？

選項：
- 用坪數或平方米
- 用一式（整間或整層）
- 不確定

ℹ️ 目的：「一式」計價難以跨廠商比價，坪數計價較透明

💬 用「一式」計價的原因可能是：數量太少、工法複雜（如跳色、特殊工序）、
   或包含多項施工。但最好還是要求提供施工面積明細。

追問：
- 如果是「一式」，有包含哪些項目？（如跳色、地板收邊打silicon、保護等）
- 每間房間的施工面積（牆+天花）大約多少坪？
```

##### **3.3 油漆工法細節（透批、批土標準）**

**問題背景**:
> 產業回饋：「要問的是廠商有沒有透批（全室批土）、批土的程度（一般平整或燈照無陰影），這是油漆價差大的原因之一。」

**新增問題**:
```
Q: 油漆工法的細節，報價有說明嗎？（這會影響品質和價格）

選項：
- 有詳細說明
- 只寫「油漆」兩個字
- 有部分說明
- 不確定

ℹ️ 目的：工法細節決定品質，如批土程度、底漆、面漆道數

💬 油漆的價差很大原因之一就是「批土程度」。一般批土只處理凹洞，
   但「透批」是全室都批土。如果要求燈照無陰影，費用會更高。

追問：
- 報價有提到「透批」（全室批土）嗎？
- 有說明批土標準嗎？（如一般平整或燈照無陰影）
- 有標示底漆幾道、面漆幾道嗎？
```

##### **3.4 壁紙去除（正確工法）**

**問題背景**:
> 產業回饋：「貼壁紙的重點應著重在壁紙與底膠的去除。壁紙去除不完整，不應用批土來平整。批土主要用於修飾表面凹洞。」

**新增問題**:
```
Q: 如果您的房子有貼過壁紙，報價有包含壁紙與底膠的「完整去除」嗎？

選項：
- 房子沒有壁紙
- 有包含壁紙去除
- 不確定是否完整去除
- 報價沒提到

ℹ️ 目的：壁紙去除不完整會造成後續脫落問題，不應用批土來掩蓋

💬 壁紙去除很重要！底紙沒去乾淨的話，不能只用批土蓋過，會造成後續脫落。
   正確做法是完整去除壁紙和底膠，才能重新上漆。
```

---

#### 🏗️ 木作工程 - 贈送項目風險（新增）

**問題背景**:
> 產業回饋：「贈送的東西要請業主特別注意，因為贈送無法要求廠商負責。如果是重要的東西，建議收費比較好，後續才有義務幫你完善。」

**新增問題**:
```
Q: 報價單有沒有「贈送」的項目？（如床框、軌道等）

選項：
- 有贈送項目
- 沒有贈送項目
- 不確定

ℹ️ 目的：了解是否有贈送項目及其風險

💬 「贈送」的東西聽起來很划算，但有個重點要特別注意：
   贈送的項目，您無法要求廠商完善或負責。
   如果是很重要的東西，建議還是改成「收費項目」，
   這樣後續才有義務幫您處理問題。

追問：
- 哪些項目是贈送的？
- 這些贈送項目對您來說重要嗎？（如果重要，建議改成收費）
```

---

#### 🏢 社區管理規定（新增）

**問題背景**:
> 產業回饋：「每個社區的自治管理辦法也有相關規定，如陽台部分不能增設鐵窗、或陽台燈色不能更改，維持大樓外觀。」

**新增問題**:
```
Q: 您的社區或大樓有特殊的施工規定嗎？

選項：
- 有，社區管委會有規範
- 沒有特殊規定
- 不確定

ℹ️ 目的：每個社區的自治管理辦法可能有不同規定

💬 有些社區會規定陽台不能加裝鐵窗、陽台燈色不能更改（維持大樓外觀）、
   或限制施工時間等。了解這些規定可以避免施工後被要求拆除。

追問：
- 社區對陽台有特殊規定嗎？（如不能加鐵窗、燈色限制等）
- 施工時間有限制嗎？（如只能平日、不能週末等）
- 需要向管委會申請施工許可嗎？
```

---

#### 📋 天花板報審提醒（優化）

**問題背景**:
> 產業回饋：「固著於構造體之天花板需申請室內裝修許可。」

**優化問題**:
```
Q: 天花板需要做造型還是平釘即可？

ℹ️ 目的：天花板固著於構造體，依法規需要報審

💬 天花板如果固著於建築結構體，依《建築物室內裝修管理辦法》
   需要申請許可。我們會協助確認是否需要報審。
```

## 🎯 新增的專業知識點

### 報價檢視要點

1. **垃圾清運費**
   - 是否明確列出？
   - 是否標示「贈送」？（建議改明訂）
   - 避免施工後才產生爭議

2. **施工保護與清潔**
   - 電梯、公共區域保護
   - 每日清潔
   - 避免被遺漏

3. **油漆價格判斷**
   - ICI 乳膠漆：約 1500/坪
   - 班傑明：約 3200/坪
   - 依品牌判斷合理性

4. **油漆計價透明度**
   - 坪數計價 > 一式計價（易比價）
   - 「一式」需問清楚包含項目

5. **油漆工法等級**
   - 一般批土：只處理凹洞
   - 透批：全室批土
   - 燈照無陰影：最高標準，費用更高

6. **壁紙去除正確工法**
   - 必須完整去除壁紙與底膠
   - 不能用批土掩蓋底紙
   - 否則會脫落

7. **贈送項目風險**
   - 贈送 = 無法要求廠商負責
   - 重要項目建議改收費
   - 確保後續有保障

8. **社區管理規定**
   - 陽台鐵窗限制
   - 陽台燈色規定
   - 施工時間限制
   - 避免施工後被要求拆除

9. **法規報審**
   - 天花板固著於結構體需報審
   - 依《建築物室內裝修管理辦法》
